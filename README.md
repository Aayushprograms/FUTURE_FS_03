# 🥭 Farmingo Alphonso Mangoes Website

## 📌 Overview
This project is a fully functional website built for a local mango business, **Farmingo Alphonso Mangoes**.

The website allows customers to browse mango varieties, calculate total price, and place orders directly via WhatsApp. It also includes an admin dashboard to manage and track orders.

---

## 🚀 Features

### 🛒 Customer Side
- Select mango varieties (Alphonso, Kesar, Badami)
- Automatic price calculation
- Order form with validation
- WhatsApp order integration
- Order confirmation popup

### 👨‍💼 Admin Side
- View all orders
- Track order status:
  - Pending
  - In Transit
  - Delivered
- Update order status dynamically

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- LocalStorage (for storing orders)
